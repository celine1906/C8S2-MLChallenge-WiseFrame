🤓 **WiseFrame**

An AI-powered iOS app that recommends eyeglass frames based on user’s face shape and skin tone, with AR try-on.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🧠 **Problem Statement**

Nara, like many users, struggles to choose eyeglass frames. She lacks clear preferences, relies heavily on opinions from others, and often feels uncertain that leading to decision fatigue and poor confidence in purchases.




💡 **Solution**

An iOS app that uses facial analysis (shape and skin tone) to recommend the best eyeglass frames. Users can also try the glasses virtually via AR.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🧩 **Key Features**

🧠 Face Analysis -	Uses Vision & ML to detect facial landmarks and define face shape

🎨 Skin Tone - Detection	CreateML-based model for skin tone detection

🤳 AR Try-On -	Render glasses using face anchor (ARKit + RealityKit)

👓 Recommendation - Suggests frames based on facial features & skin tone

🛠 **Tech Stack**

CreateML & Python → CoreML – Model training

Vision – Face landmark detection

ARKit + RealityKit – AR rendering on face

SwiftUI – UI design

------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🧪 **Design Thinking Process**
1. Conducted interviews with glasses users to identify pain points
2. Built user persona & journey
3. Iterated based on feedback
4. Focused on reducing decision fatigue with personalized suggestions
